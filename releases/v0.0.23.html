üöÄ Release v0.0.23

Week of November 10, 2025

Portal OS desktop workspace, dedicated release hub, and a Tribes-inspired Zero-G Ski Range tune-up

Overview

This milestone introduces the Portal OS desktop as the new jumping-off point for contributors and launches a dedicated release hub so every build has a permanent changelog home. Gameplay work stayed intentionally scoped to solo polish while we concentrate on the foundational workspace, contacts, CRM, and finance flows.

Desktop Home Workspace

  • Launched the Portal OS desktop with status bar quick links, a live clock, and a launcher that routes directly to calendar, tasks, chat, and the broader portal.
  • Added keyboard-friendly navigation, ARIA roles, and layered window chrome so multiple apps can sit side-by-side with future GunDB-backed state persistence.
  • Wired the scripted workspace so shortcuts, launcher pins, and system buttons open their targets consistently across desktop, tablet, and headset browsers.
  • Surfaced quick links into contacts, CRM, and finance views directly from the dock to speed up the workflows we plan to deepen next.

Release Hub & Timeline

  • Launched the release section as a central directory so every weekly milestone from v0.0.1 onward has a consistent layout, metadata, and navigation.
  • Linked each release with previous/next controls, enabling quick comparisons between v0.0.22 stability work and the new desktop-centric v0.0.23 milestone.

Zero-G Ski Range (Tribes Flight Prototype)

  • Debuted the arena with energy bars, weapon switching, target integrity readouts, and slope physics that reward chaining skis with jetpack bursts.
  • Implemented adaptive canvas sizing, orientation handling, and safe-area aware HUD panels for phones and tablets.
  • Tuned weapon spread, projectile drag, and target regeneration delays to keep the single-player loop responsive while we iterate on balance before layering in any cooperative modes.

Portal & Access

  • Highlighted the new experience across the Game Hub and portal navigation so testers can jump straight into the prototype from the desktop launcher.
  • Ensured cross-links from the desktop home to legacy pages (Start, Portal classic) so contributors can compare the new workspace against existing flows.

Iteration Notes

  • de34d44: Initialised the Zero-G Ski Range with the first pass of physics, HUD widgets, and portal discovery links.
  • be053e4: Reworked layout scaling, safe area padding, and coarse pointer detection to make the arena comfortable on phones and tablets.
  • 5b2138d: Polished touch responsiveness, pointer capture fallbacks, and action clustering so taps translate into predictable momentum changes.

Known Issues

  • Contacts (contacts/index.html) does not yet support editing existing entries.
  • The CRM (crm/index.html) mirrors the same limitation and requires a future patch.

Looking Ahead

  • Continue refining the home workspace layout for tablet and headset users, layering in more productivity widgets.
  • Add editable detail panes to contacts and CRM records so teams can update client information in place.
  • Expand finance tracking to make revenue, expenses, and invoices easier to monitor.
  • Polish the Tribes Flight gameplay loop with smoother handling, clearer goals, and richer solo challenges.
