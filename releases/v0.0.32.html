Release v0.0.32

Week of December 29, 2025

Overview

The portal focused on discoverability and builder productivity this week, with a refreshed landing flow, new workbench hubs, and fresh labs covering XR motion, decentralized cloud, and local model setup.

Portal quick links

  • Portal home – jump into every workspace, lab, and experiment.
  • Release hub – browse weekly milestones and product checkpoints.
  • OpenAI Workbench – build, preview, and ship with shared defaults.
  • Web Builder – create web experiences with guided launch flows.
  • XR Motion Lab – prototype VR/AR motion controls from the browser.
  • Decentralized Cloud – explore the open cloud roadmap and architecture.
  • Local Models – track on-device setup notes and experiments.

Landing page refresh

  • Refocus the portal home hero on app discovery with updated layout and messaging.
  • Retire the breathing demo headline link so the primary CTA stays on exploration and launch.
  • Highlight new app cards for local models and the SEO planning workspace to keep the landing grid current.

Workbench workflow hubs

  • Add setup, preview, and logs hubs inside the OpenAI Workbench so builders can navigate the full flow without losing context.
  • Improve workbench shared default key management with clearer UI guidance and stronger hints.
  • Smooth mobile editing and repo creation experiences with ergonomic layout updates.

New labs and directories

  • Launch the XR Motion Lab prototype for accelerometer- and camera-driven VR/AR control exploration.
  • Introduce the Decentralized Cloud landing page and directory structure.
  • Move TinyMark into its own home and link it from the portal navigation.
  • Add a self-hosted deployment guide for GitHub and Vercel setups.

Ideas Lab audience testing

  • Add audience-focused experiment pages in the Ideas Lab to support future A/B testing across the concept portfolio.
