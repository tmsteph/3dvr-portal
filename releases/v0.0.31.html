Release v0.0.31

Week of December 22, 2025

Overview

The new Ideas Lab launches with trackable experiments, the portal gains a project directory for active explorations, and shared defaults across the AI builders pick up clearer vault imports plus rate limits for default keys.

Portal quick links

  • Portal home – hop across workspaces, labs, and the release hub.
  • Release hub – browse weekly milestones and product checkpoints.
  • Projects directory – see active explorations and next steps to ship.
  • Ideas Lab – test landing experiments with local view and CTA stats.
  • OpenAI Workbench – code, preview, and deploy with shared defaults.
  • Web Builder – generate and publish sites with guarded default keys.
  • Admin – manage shared defaults, score rewards, and payouts.

Ideas Lab experiments

  • Launch an Ideas Lab dashboard with per-card stats for page views, CTA clicks, last-seen timestamps, and a manual reset that stores data in localStorage per landing page.
  • Add three idea landers—Websites, Systems, and Tech Support—with prefilled mailto CTAs that include the idea name and URL while logging CTA clicks per visitor.
  • Keep portal navigation close at hand so visitors can hop back to home or review other experiments without losing their local stats.

Project directory & navigation

  • Introduce a Projects directory that highlights the wearable input kit exploration, spells out the GunJS data paths for concepts and handoff artifacts, and links to shared notes and tasks.
  • Surface new Projects and Ideas Lab entry points on the portal home so contributors can find experiments and explorations alongside existing tools.
  • Retain dark-theme readability with mobile-friendly grids and skip links that jump directly to project content.

Workbench builders & shared defaults

  • Guard the rate-limited Web Builder by tracking shared default usage per billing tier and identity before generating or publishing sites, with Gun-backed counters that reset daily.
  • Harden OpenAI Workbench defaults by aligning cipher fields for OpenAI, Vercel, and GitHub tokens so shared keys unlock reliably across vault imports and passphrase hints.
  • Let admins import their saved workbench secrets—or vault backups when needed—directly into the shared defaults form before re-encrypting them for the team.
