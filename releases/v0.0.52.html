Release v0.0.52

Week of June 29, 2026

This release gets the portal current for the Friday, July 3, 2026 decision meeting and the Monday, July 6, 2026 production release. The focus is narrower: make the free path obvious, keep sign-in from trapping returning users, and document the next money path before building more billing machinery.

  • Free-first portal and Monday release path: Friends & Family Pass launched in PR #907, then the free offer copy was tightened in PR #914, PR #921, and PR #922. The portal now points first-time people toward Daily Direction and one small next step instead of requiring them to understand the whole app dock.
  • Daily Direction, guide mode, and owned app drafts: portal home gained the prompt guide in PR #926, real AI-backed routing in PR #927, a simpler Daily Direction app in PR #928, automatic portal identity in PR #929, and user-owned app draft controls in PR #930.
  • Forge and visual exploration: 3DVR Forge became more guided in PR #923, more focused on life direction in PR #924, and more app-like in PR #925. The portal also added the String Theory Visualizer in PR #931.
  • Sign-in and cache recovery: the visible sign-in links came back in PR #938, the friends/family invite tightened in PR #939, normal sign-in links were fixed in PR #940, and the stale identity/cache sequence was hardened across PR #941 through PR #947.
  • Roadmap and billing decision docs: 3DVR Long-Term Plan and Roadmap landed in PR #976. The No-Account Stripe Payment Plan landed in PR #977 so the next billing move can be simple: plan card, Stripe link, receipt, optional portal setup.
