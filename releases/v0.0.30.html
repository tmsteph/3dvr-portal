Release v0.0.30

Week of December 15, 2025

Overview

Score governance and payout math get sharper this week. The live point-to-dollar dashboard now estimates value per person directly from Stripe balances, while the admin console picks up dedicated score deductions with caps to prevent steep drops. The Score system itself enforces bounded decrements so profile totals stay resilient during audits and retroactive fixes.

Portal quick links

  • Portal home – hop across workspaces and the release hub.
  • Release hub – browse every weekly milestone.
  • Points – review reward structures and leaderboard signals.
  • Point-to-dollar ratio – see live pool math, cashout previews, and per-person value estimates.
  • Admin – search profiles, award or deduct points, and manage rewards.
  • Sales – keep score cards, top performers, and contact follow-ups in view.
  • CRM & contacts – log touches, schedule follow-ups, and track the weekly challenge.

Points, payouts, and transparency

  • Surface a "Dollar value per person" table on the point-to-dollar ratio page so contributors can see estimated cash value alongside points.
  • Keep live pool, leaderboard totals, and per-person estimates synchronized with Stripe and Gun-backed scores for accurate cashout previews.
  • Clarify the path back to points overview while preserving refresh controls for real-time calculations.

Score governance and safeguards

  • Add admin-only score deduction controls with helper messaging so leaders can correct totals without manual edits.
  • Apply decrement caps and floors inside the Score system to guard against steep drops and keep leaderboard math consistent across devices.
  • Retain admin search and selection flows, making it easy to grant or deduct points against the correct profile.

Reliability & navigation

  • Maintain dark-theme readability and responsive spacing for the release hub, admin console, and points ratio dashboard on desktop and mobile.
  • Keep breadcrumb-style navigation between releases and the broader portal so reviewers stay oriented during audits.
