Release v0.0.49

Week of June 8, 2026

This weekly release tightens play, capture, and validation surfaces while turning a set of portal strategy notes into clearer operating language.

  • Games and flight controls: Stellar Drift gained keyboard flight, spacebar firing, tuned arrow aim, and random enemies while Games, Pong Arena, and the Zero-G Ski Range experience moved toward sharper desktop play.
  • Capture and validation: CRM conversation capture, simplified capture flows, the Project Desk portal relationship, and problem-validation docs made the start path more concrete.
  • Portal operating language: the cybernetic portal, civic operating system, worker power, portal vision card, and agent execution notes clarified what the portal is for before adding more layers.
  • Practice and prototypes: Manifestation Practice, 3DVR Girl, and the customer-facing start-page pass gave experimental work clearer homes.
