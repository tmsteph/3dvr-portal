Release v0.0.50

Week of June 15, 2026

This release brings the portal back onto the current PR stream, with a complete Launch Site publish path, cleaner access surfaces, and fewer rough edges on experimental apps.

  • Launch Site publishing: Launch Site moved from a simple deploy path to a tested production workflow: 3dvr team deployments, readiness polling, domain registration, DNS verification, and custom-domain alias handling.
  • Portal access cleanup: guest entry, app search, home backlinks, the hero tagline, and the removed botanical border all moved the portal toward faster first-party access.
  • New and refined work surfaces: Sober Spark, Seed Deck, Spatial VR Portal, and CRM conversation saves landed or became more usable.
  • Admin and defaults: shared builder defaults and admin key preservation now avoid wiping working configuration when only one value changes.
