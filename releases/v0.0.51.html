Release v0.0.51

Week of June 22, 2026

This release catches the portal up from the post-v0.0.50 PR stream: mobile reliability, CRM install polish, the Human O.S. / Workshop switch, Forge becoming a real product route, and the first paid sprint revenue paths.

  • Portal shell and mobile stability: the homepage overflow fix from PR #843, smaller desktop hero buttons from PR #877, the Games hero feature from PR #848, and CRM PWA/install improvements from PR #823 through PR #847 made the portal easier to open and trust.
  • Human O.S., rooms, and launch flow: Purpose Movement, Launch Room, Reality Builder, Fractal Explorer, Orbital Courier, and Pocket Window expanded the room system while the portal gained the Human O.S. filter in PR #878 and defaulted back to Workshop in PR #882.
  • Connect, fascia, and 3DVR Girl: 3DVR Connect and Fascia Release shipped and tightened across PR #746 through PR #756, while 3DVR Girl gained guide selection, portal-section polish, and rescued gallery assets in PR #787, PR #788, and PR #898.
  • Money Printer and paid sprint paths: money-printer, Offer Audit, 3DVR Forge, Forge Revenue Sprint, Portfolio Validation Sprint, and Growth Operator now form a practical loop: generate a brief, convert it into a paid offer, capture fit-checks, and import those leads into the operator queue.
  • Forge becomes the product: 3DVR Forge moved from idea to route in PR #883, gained live AI behavior in PR #884, simplified its prompt flow in PR #888, and picked up revenue-focused guidance and sprint CTAs through PR #895 through PR #903.
