Release v0.0.41

Week of April 13, 2026

This weekly release centers on five concrete shifts:

  • Pocket Workstation: a new lightweight workspace lands in the portal.
  • Life OS: the Life app moves into a clearer daily operating shape.
  • CRM + billing handoff: follow-through gets tighter across customer flows.
  • Portal home: the front door moves closer to a real command center.
  • 3dvr-agent: companion outreach tooling starts taking shape.

Overview

This week pushes 3DVR further toward a practical operating system for real daily use.

  • The portal adds a new portable workspace, stronger CRM and billing follow-through, and a more direct command-center front door.
  • The live surface gets a lightweight issue launcher so bugs and follow-up can be captured without leaving the product context.
  • Committed 3dvr-agent work starts shaping a local outreach pipeline for crawl, enrich, tracking, and message iteration.

Portable workspaces and operator surfaces

  • Add the new Pocket Workstation app and pairing flow so the portal has a clearer lightweight mobile desk.
  • Make the portal launcher feel more like a command center with stronger layout, hierarchy, and app-entry cues.
  • Add a GitHub issue launcher across portal pages so bugs and follow-up are easier to capture from the live surface.
  • Bring the calendar month view forward on mobile and make the Logic Lab drill card easier to activate.

Customer journey, CRM, and billing

  • Move the free-trial email field higher and tighten homepage billing handoff so the entry path is more explicit.
  • Simplify the portal customer journey and keep Stripe plan-switch handling aligned with portal origin rules.
  • Normalize contacts into CRM handoff records and move primary CRM and contacts actions to the top of those pages.
  • Send email notifications for Stripe subscription updates so billing changes create a clearer operator trail.

Life OS and companion agent work

  • Refocus the Life app into Life OS with a clearer daily operating shape and updated tests around that flow.
  • Keep finance visibility, billing, and outreach work tied to revenue-first operations instead of generic productivity.
  • On committed 3dvr-agent history this week, add a local sales agent system plus crawl, enrich, tracking, and A/B outreach message scripts.
  • Treat the agent repo as companion operator tooling: it sharpens outreach execution while the portal keeps becoming the shared front door and workflow desk.
