Release v0.0.43

Week of May 4, 2026

This weekly release centers on the new May agent beta and inbox hardening:

  • 3dvr-agent: the May beta is tagged as 1.0.1-beta.2 and aligned to the new weekly cut.
  • Inbox monitoring: delivery failures now get detected, surfaced, and marked as failed leads.
  • Portal cadence: the weekly release hub now includes the new May 4, 2026 entry.

Overview

This release keeps the portal and agent aligned on the same weekly rhythm while tightening failure handling for outbound email.

  • The agent release moves from the April pre-release to a new May beta tag.
  • The inbox monitor now watches for bounce mail and updates leads that fail delivery.
  • The release hub keeps the portal cadence visible for the current week.

Agent release

  • Tag and publish the new beta for the week of May 4, 2026.
  • Keep the release notes explicit about the May rollout.

Delivery monitoring

  • Detect bounce mail in the inbox monitor.
  • Mark leads as failed when their delivery bounces.
