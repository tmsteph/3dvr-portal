Release v0.0.37

Mid February 2026

Jetpack Corridor gets a serious gameplay pass while Playwright smoke checks and Money Autopilot push the portal toward something that can test, automate, and publish with more confidence.

Overview

Mid-February folds together three different threads of work: the arcade gets a stronger mobile-first Jetpack experience, the portal finally gains automated browser smoke checks, and Money AI evolves from a single page into an automation loop with secure tokens, publishing routes, and cron support.

Jetpack and game polish

  • Integrate Jetpack Corridor into the games hub and keep mobile navigation workable with a collapsible top nav.
  • Refactor the game into modules, harden boot and fallback loading, and tune obstacles and start flow.
  • Improve the mobile HUD, joystick strafe, vertical climb, and touch-driven turn behavior.

Testing and runtime stability

  • Add automated Playwright smoke checks for local and CI verification.
  • Fix cross-platform scheduler and test stability issues so browser coverage is more reliable.
  • Keep the sales homepage links aligned with the new dedicated pitch page.

Money automation stack

  • Add the Money Autopilot loop with Gun sync, secure trigger support, publishing paths, and market discovery.
  • Introduce per-user autopilot tokens, plan rate limits, and stronger token-secret fallback handling.
  • Wire checkout CTAs and cron scheduling while fitting within Hobby-plan runtime limits.
  • Tighten signal relevance and keyword sanitization so the automation loop stays closer to real demand.
