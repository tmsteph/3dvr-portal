Release v0.0.34

Week of January 5, 2026

Overview

This release tightens meeting workflows, introduces a Supabase testing entry point, and rolls out a brand-new partner grid plus fresh meal planning and shopping list experiences across the wider ecosystem.

  • Portal home – jump into every workspace, lab, and experiment.
  • Release hub – browse weekly milestones and product checkpoints.

  • CRM – track leads, notes, and outreach in one place.

Meeting notes upgrades

  • Enable inline editing for meeting titles and attendee names to keep session records accurate.
  • Stabilize meeting note sync behavior so shared updates stay consistent across sessions.

Platform verification

  • Introduce a Supabase test app entry point to validate configuration and environment connectivity.
  • Document test guidance in the Supabase test suite so contributors can validate credentials quickly.

tmsteph.com redesign updates
